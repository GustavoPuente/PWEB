Exibição das Três Formas de Criar o Objeto

Forma 1: Notação de Objeto Literal

const Funcionario1 = {
  Matricula: 12345,
  Nome: "João Silva",
  Função: "Desenvolvedor"
};

Forma 2: Usando Construtor de Objeto

const Funcionario2 = new Object();
Funcionario2.Matricula = 12345;
Funcionario2.Nome = "João Silva";
Funcionario2.Função = "Desenvolvedor";

Forma 3: Usando Função Construtora

function Funcionario(matricula, nome, funcao) {
  this.Matricula = matricula;
  this.Nome = nome;
  this.Função = funcao;
}

const Funcionario3 = new Funcionario(12345, "João Silva", "Desenvolvedor");
